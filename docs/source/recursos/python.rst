Python
======

Python será uno de los lenguajes principales utilizados en el curso.

Comprobar instalación
---------------------

.. code-block:: bash

   python --version

Crear un entorno virtual
------------------------

.. code-block:: bash

   python -m venv .venv

Activar en Windows
------------------

.. code-block:: powershell

   .\.venv\Scripts\Activate.ps1

Taller integrador — Experta
===========================

Descripción
-----------

El taller integrador busca aplicar los conceptos del curso mediante el desarrollo
de una solución basada en inteligencia artificial y herramientas especializadas.

Componentes preliminares
------------------------

* Gestión de hechos y reglas.
* Uso de herramientas o funciones especializadas.
* Manejo de agenda y conflictos.
* Priorización mediante salience.
* Trazabilidad de las decisiones.

.. note::

   Esta página será ampliada con el enunciado, los casos de prueba y la rúbrica.

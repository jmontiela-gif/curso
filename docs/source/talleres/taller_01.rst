Taller 1 — Sistemas expertos
============================

Descripción
-----------

En este taller se desarrollará un sistema experto basado en reglas para resolver
un problema definido.

Objetivos
---------

* Representar conocimiento mediante hechos y reglas.
* Implementar un motor básico de inferencia.
* Registrar la trazabilidad del razonamiento.
* Validar la solución mediante pruebas automatizadas.

Taller 1 — Sistemas expertos
============================

.. rst-class:: workshop-label

   REPRESENTACIÓN DEL CONOCIMIENTO

Descripción
-----------

En este taller el estudiante diseñará e implementará un sistema experto
basado en reglas.

La solución deberá representar el conocimiento de un dominio mediante hechos,
reglas y un mecanismo de inferencia capaz de obtener conclusiones a partir
de la información disponible.

Objetivos de aprendizaje
------------------------

Al finalizar el taller, el estudiante estará en capacidad de:

* Diferenciar hechos, reglas y conclusiones.
* Representar conocimiento mediante reglas ``si-entonces``.
* Implementar un mecanismo de inferencia.
* Explicar la secuencia de razonamiento utilizada por el sistema.
* Validar el comportamiento mediante casos de prueba.

Conceptos previos
-----------------

* Representación del conocimiento.
* Lógica proposicional.
* Reglas de inferencia.
* Modus ponens.
* Encadenamiento hacia adelante.
* Bases de hechos y bases de conocimiento.

El material del curso presenta los sistemas expertos como programas que
simulan el razonamiento y la toma de decisiones de un experto humano,
utilizando hechos, reglas e inferencias sobre un dominio específico.


Estructura del taller
---------------------

.. container:: workshop-steps

   **1. Comprensión del problema**

   Identificar el dominio, los actores, las entradas y las decisiones que deberá
   producir el sistema.

   **2. Diseño de la base de conocimiento**

   Definir los hechos y las reglas necesarias para representar el dominio.

   **3. Implementación**

   Construir el sistema experto utilizando Python.

   **4. Trazabilidad**

   Registrar las reglas activadas y las conclusiones alcanzadas.

   **5. Validación**

   Comprobar el funcionamiento mediante pruebas automatizadas.
